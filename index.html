⭐ Featured Project
AI / Healthcare

BreastCare AI

A multi-layer breast cancer risk estimation prototype combining ensemble ML models (XGBoost + LightGBM), SHAP explainability, RAG-powered Q&A over curated BI-RADS medical documents, and LLM-grounded explanations — all wrapped in a safety filter that prevents diagnostic language.

XGBoost LightGBM SHAP FAISS RAG Gemini AI Streamlit Plotly
View Source on GitHub
🏗️ System Architecture ▼

Takes 4 mammographic features (age, shape, margin, density) through a 5-layer pipeline:

① ML Layer: LightGBM → malignancy probability
② XAI Layer: SHAP TreeExplainer → feature attributions
③ RAG Layer: FAISS over 60 curated BI-RADS chunks
④ LLM Layer: Gemini AI → grounded explanation
⑤ Safety Layer: Regex filter strips diagnostic language
